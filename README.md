# EventBuzz
This is the project on which the associate team is working on.


### How to start 
``` bash 
git clone https://github.com/GDG-On-Campus-BVM/EventBuzz.git
cd EventBuzz/
```

#### Start Backend 
- conform you are at EventBuzz folder 
``` bash 
cd server/
yarn 
yarn dev
```

#### Start Frontend 
- Open new terminal 
``` bash 
cd client/
yarn start
```
